# macOS Keyboard & Productivity Shortcuts (20)

## Everyday editing (the ones I use constantly)

1. **Cut** — `⌘ X`
2. **Copy** — `⌘ C`
3. **Paste** — `⌘ V`
4. **Undo** — `⌘ Z`
5. **Redo** — `⇧ ⌘ Z`
6. **Select all** — `⌘ A`
7. **Find in page / document** — `⌘ F`
8. **Find next** — `⌘ G`
9. **Find previous** — `⇧ ⌘ G`

## Working faster across apps & windows

10. **Switch apps** — `⌘ Tab` (hold ⌘ and tap Tab to cycle)
11. **Close current window** — `⌘ W`
12. **Quit the current app** — `⌘ Q`
13. **Force Quit (if an app freezes)** — `⌥ ⌘ Esc`

## Search + navigation (massive time savers)

14. **Spotlight search** — `⌘ Space`
15. **Mission Control (see all windows/spaces)** — `Control ↑`
16. **Show Desktop** — `⌘ Mission Control (F3)` _(depending on Mac/keyboard settings)_

## Screenshots / recordings

17. **Screenshot entire screen** — `⇧ ⌘ 3`
18. **Screenshot a selected area** — `⇧ ⌘ 4`
19. **Screenshot toolbar / screen recording tools** — `⇧ ⌘ 5`

## Quick safety / privacy

20. **Lock screen immediately** — `Control ⌘ Q`
